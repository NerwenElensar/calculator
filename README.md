# Calculator <img src="https://d29fhpw069ctt2.cloudfront.net/icon/image/84736/preview.svg" alt="drawing" width="40"/>

Yup a calculator. Just like you know it. Maybe not as complex as yours from your school days, but if you need some basic functions like adding and subtracting numbers(also some multiplications or divisions). Here it is my friend. 


### Learning Outcome 💻

The calculator should take a pair of numbers at one time and return the result. Then the result is used as the first number of the next operation. Pressing `=` evaluates two numbers and shows the result. The `Clear` button wipes the screen and let's you start from scratch. 
**The Calculator does not evaluate more than two numbers at a time**

#### Other 
- Planning out an algorithm beforehand answering the following questions among others:
  - Which values should be stored and how? 
  - When does the calculator actually calculate a result?
  - What happens when the user presses the equals button, before having a second operand?
  - What happens when the user divides by zero?
